Dependencies:

-compton
Compositor for screen tearing, works 70% of the time

-polybar
Taskbar configured with multiple system readouts

-nitrogen
Wallpaper selector, the actual wallpaper itself needs to be selected manually 

-rofi
Dmenu replacement, configure theme with rofi-theme-selector

-lxappearance
Not required, but I use it to change the theme of apps

-pulseaudio
The audio server that i3 and polybar are configured to use

-pavucontrol
Not required, it's a GUI for pulseaudio

Other information:

These files are configured for the base version of i3 without gaps. The system readouts in polybar are configured for a Dell Latitude 3490.
